Patrick is currently an **unsigned Windows desktop app** (the alpha).

## Download

Grab the latest installer from the
[releases page](https://github.com/mhurhangee/patrick/releases/latest), then run
it. On first launch, Windows SmartScreen may warn "Windows protected your PC" —
click **More info → Run anyway**. (Code signing comes with the beta.)

## Bring your own key

Patrick uses **your** AI provider key — Anthropic, OpenAI, or Google. Your key
stays on your machine and talks only to the provider you choose; you pay that
provider directly for usage, usually pennies per response.

Add your key in a **profile** inside the app:

```text
Profile → AI → paste your provider key
```
