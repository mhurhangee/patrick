Patrick is one agent that works **inside your documents** through tools — it reads
the live document, finds text, and proposes **native Word tracked changes**.

## Human in the loop

Patrick **proposes**; you decide. Edits arrive as tracked changes you accept or
reject. Bigger moves — opening a document, creating a draft, saving a note — come
as **accept/reject cards**. Nothing happens behind your back.

## Transparent by default

For every turn you can see:

- the **system prompt** it was given (yours to read and edit),
- its **reasoning** and every **tool call**,
- the **tokens and cost**.

## One Patrick

There's one Patrick per chat, with one set of instructions frozen for that
conversation. Start a new chat to change the instructions.
